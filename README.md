
# Freelanzo

## Description:
Freelanzo is a freelance platform that connects clients with skilled freelancers in various fields such as programming, design, and content creation. Users can post projects, submit proposals, communicate easily within the platform, and manage the entire workflow including collaboration and secure payments in one place.



## Features


- User authentication and role-based access (Client / Freelancer).
- Profile creation and management for both clients and freelancers.
- Freelancer portfolio with projects and images.
- Project posting with budget, deadline, and required skills.
- Proposal submission and management.
- Real-time messaging between users and admin.
- Search and filtering by skills and categories.
- Project tracking (open, in progress, completed, submitted).
- Work submission via file upload or external link.
- Review and rating system for both clients and freelancers.
